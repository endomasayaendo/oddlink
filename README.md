# OddLink

A URL shortener that generates surreal English phrases as short codes.

Instead of `https://example.com/abc123`, you get something like:
`http://localhost:8080/golden-clock-whispers-to-silent-butterfly`

## Tech Stack

| Layer | Tech |
|-------|------|
| Backend | Spring Boot 3.5.9, Java 21, Spring Data JPA |
| Frontend | React 19.2, TypeScript, Vite, React Router 7, Recharts |
| Database | PostgreSQL 17 (Docker) + Flyway |

## Prerequisites

- Java 21
- Node.js 18+
- Docker + Docker Compose

## Getting Started

### 1. Configure environment variables

```bash
cp .env.example .env
```

Edit `.env` as needed (defaults work for local development):

```env
DB_NAME=oddlink
DB_USERNAME=postgres
DB_PASSWORD=postgres
```

### 2. Start the database

```bash
docker compose up -d
```

### 3. Start the backend

```bash
./mvnw spring-boot:run
```

The API runs on `http://localhost:8080`.

### 4. Start the frontend

```bash
cd frontend
npm install
npm run dev
```

The app runs on `http://localhost:3000`.

## API Reference

### Shorten a URL

```
POST /api/issue
Content-Type: application/json

{
  "originalUrl": "https://example.com/some/long/url"
}
```

**Response:**

```json
{
  "shortUrl": "http://localhost:8080/golden-clock-whispers-to-silent-butterfly",
  "originalUrl": "https://example.com/some/long/url"
}
```

### Redirect

```
GET /{shortCode}
```

Redirects to the original URL.

### Analytics

```
GET /api/analytics/{shortCode}
```

**Response:**

```json
{
  "shortCode": "golden-clock-whispers-to-silent-butterfly",
  "shortUrl": "http://localhost:8080/golden-clock-whispers-to-silent-butterfly",
  "originalUrl": "https://example.com/some/long/url",
  "totalAccessCount": 42,
  "createdAt": "2025-01-01T00:00:00",
  "expiresAt": "2025-04-01T00:00:00",
  "dailyAccess": [
    { "date": "2025-01-01", "count": 10 },
    { "date": "2025-01-02", "count": 32 }
  ]
}
```

## Phrase Generation

Short codes are generated by combining words from four categories — adjectives, nouns, verbs, and adverbs — into one of six surreal patterns:

| Pattern | Example |
|---------|---------|
| `[adj]-[noun]-[verb]-to-[adj]-[noun]` | `golden-clock-whispers-to-silent-butterfly` |
| `[noun]-of-[adj]-[noun]-[verb]` | `river-of-broken-mirror-melts` |
| `[adj]-[noun]-[verb]-with-[adj]-[noun]` | `silent-cat-dances-with-lost-moon` |
| `[verb]-[adj]-[noun]-into-[noun]` | `fold-ancient-light-into-clock` |
| `[adj]-[noun]-[adv]-[verb]` | `hollow-tree-slowly-remembers` |
| `[noun]-[adv]-[verb]-[adj]-[noun]` | `smoke-gently-carries-blue-key` |

Words are seeded into the database via Flyway migrations and cached in memory at startup.

## Running Tests

```bash
./mvnw test
```

Tests use H2 in-memory database and MockMvc.

## Environment Variables

| Variable | Default | Description |
|----------|---------|-------------|
| `DB_NAME` | `oddlink` | PostgreSQL database name |
| `DB_USERNAME` | `postgres` | PostgreSQL username |
| `DB_PASSWORD` | `postgres` | PostgreSQL password |
| `BASE_URL` | `http://localhost:8080` | Base URL used in generated short links |
| `FRONTEND_URL` | `http://localhost:3000` | Frontend origin (used for CORS) |
| `CORS_ALLOWED_ORIGINS` | `http://localhost:3000` | Allowed CORS origins |
